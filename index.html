GlueDarwin
A lightweight desktop and server operating system.

Open desktop computing, inspired by Darwin.

GlueDarwin is a fast and practical operating system designed for both desktop and server environments. It combines a lightweight userland, a familiar workflow and an open development model focused on usability and performance.

Not XNU

GlueDarwin is not based on Apple's XNU kernel. It follows its own direction while preserving the openness, flexibility and accessibility that made Darwin interesting to developers.

An Open Future

Modern macOS is a proprietary operating system. GlueDarwin explores a different path: an open platform inspired by Darwin's heritage, developed in public and available to everyone.
